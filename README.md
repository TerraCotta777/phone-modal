# phone-modal

> A reusable React service to set user's phone number with modal UI.

## Install

```bash
npm install phone-modal
```
or
```bash
yarn add phone-modal
```

## Usage

```tsx
import { setPhone } from 'phone-modal';

await setPhone();
```

## Features

- Asynchronous setPhone API
- Modal-based UI
- LocalStorage saving
- Lightweight bundle
- React + TypeScript
